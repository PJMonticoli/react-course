# react-course

## Creating a New Application

1. **Initialize a Project with npm:
    ```sh
    npm init -y
    ```
2. **Create a New Application with Vite:
    ```sh
    npm create vite@latest
    ```
3. **Get Started:
    ```sh
    npm create vite@latest
    ```
4. **Install Dependencies:
  ```
  npm install
  npm run dev
 ```
